Здесь лежат исследовательские ноутбуки (с русскими пояснениями).

Доступные ноутбуки:
- `01_eda.ipynb` — первичный EDA по реальным строительным данным;
- `02_real_data_training.ipynb` — обучение на реальном датасете;
- `03_shap_analysis.ipynb` — SHAP-интерпретация факторов;
- `04_graph_scheme.ipynb` — графовая схема строительного проекта;
- `05_training_plots.ipynb` — графики результатов обучения для статьи.

Примечание:
- Если у тебя пока нет реальных файлов, используй шаблонные названия из `data/raw/README.md` и затем подмени своими CSV.
